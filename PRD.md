# 密碼產生器工具 - 產品需求文檔

立即產生強大且安全的密碼，保護您的線上帳戶安全。

**體驗品質**: 
1. 簡潔專業 - 介面乾淨直觀，專注於核心功能
2. 即時回饋 - 密碼強度即時顯示，操作反應迅速
3. 安全可靠 - 本地生成密碼，不儲存敏感資料

**複雜度等級**: 
微型工具（單一目的）- 專注於密碼生成這一核心功能，提供必要的自訂選項

## 核心功能

### 密碼生成
- 功能：根據使用者設定產生隨機密碼
- 目的：提供強大且符合需求的密碼
- 觸發：頁面載入時自動生成，或點擊重新生成按鈕
- 流程：設定參數 → 點擊生成 → 顯示密碼 → 顯示強度評估
- 成功標準：生成符合所選條件的隨機密碼

### 長度調整
- 功能：滑桿控制密碼長度（8-64字元）
- 目的：滿足不同服務的密碼長度要求
- 觸發：拖拽滑桿或點擊+/-按鈕
- 流程：調整長度 → 即時顯示數值 → 自動重新生成密碼
- 成功標準：密碼長度準確反映設定值

### 字元類型選擇
- 功能：勾選框控制包含的字元類型（大寫、小寫、數字、符號）
- 目的：符合各種密碼政策要求
- 觸發：點擊勾選框
- 流程：切換選項 → 即時重新生成 → 更新密碼強度
- 成功標準：生成的密碼僅包含已選擇的字元類型

### 密碼複製
- 功能：一鍵複製生成的密碼到剪貼板
- 目的：方便使用者快速使用密碼
- 觸發：點擊複製按鈕
- 流程：點擊複製 → 密碼寫入剪貼板 → 顯示成功提示
- 成功標準：密碼成功複製，顯示確認訊息

### 密碼強度評估
- 功能：即時評估並顯示密碼強度
- 目的：幫助使用者了解密碼安全性
- 觸發：密碼生成或參數變更時自動評估
- 流程：分析密碼 → 計算強度分數 → 顯示評級和顏色
- 成功標準：準確顯示密碼強度等級（弱/中等/強/非常強）

## 邊緣情況處理
- **無字元類型選擇**: 強制至少選擇一種字元類型
- **極短密碼**: 最小長度限制為8字元
- **複製失敗**: 顯示錯誤提示並提供手動選擇文字的備選方案
- **瀏覽器不支援**: 提供基本功能降級方案

## 設計方向
介面應感覺現代、專業且值得信賴 - 簡潔的設計傳達安全感，充足的留白和清晰的視覺層次引導使用者注意力到關鍵功能。

## 色彩選擇
補色配色方案 - 使用藍色作為主色調傳達信任和專業，搭配橙色/綠色強調重要操作和狀態。

- **主色調**: 藍色 oklch(0.6 0.15 240) - 傳達信任、安全和專業
- **輔助色彩**: 淺灰色背景，深灰色文字
- **強調色**: 綠色 oklch(0.7 0.15 140) - 用於成功狀態和強密碼指示
- **前景/背景配對**: 
  - 背景（淺灰 oklch(0.98 0 0)）：深色文字 oklch(0.2 0 0) - 對比度 15.8:1 ✓
  - 主色（藍 oklch(0.6 0.15 240)）：白色文字 oklch(1 0 0) - 對比度 4.9:1 ✓
  - 強調（綠 oklch(0.7 0.15 140)）：深色文字 oklch(0.2 0 0) - 對比度 5.2:1 ✓

## 字體選擇
使用 Inter 字體傳達現代感和可讀性，支援多語言字元顯示。

- **字型層級**: 
  - H1（主標題）: Inter Bold/32px/緊密字間距
  - H2（副標題）: Inter Medium/18px/標準字間距
  - Body（內文）: Inter Regular/16px/標準字間距
  - Label（標籤）: Inter Medium/14px/標準字間距

## 動畫效果
微妙的功能性動畫增強使用體驗 - 按鈕懸停效果、密碼生成時的淡入動畫、強度指示器的顏色轉換。

- **功能意義**: 動畫傳達應用程式的可靠性和響應性，引導使用者注意狀態變化
- **動作層次**: 主要動畫用於密碼生成和複製確認，次要動畫用於懸停狀態

## 組件選擇
- **組件**: Button、Card、Slider、Checkbox、Badge 來自 shadcn，使用自訂樣式符合安全工具美學
- **自訂組件**: PasswordStrengthIndicator（強度顯示）、PasswordDisplay（密碼顯示區域）
- **狀態**: 按鈕有明確的懸停、點擊、禁用狀態，勾選框有清晰的選中/未選中視覺差異
- **圖示選擇**: Phosphor icons - Copy、RotateRight、Plus、Minus 等簡潔圖示
- **間距**: 使用 Tailwind 的 4、6、8、12 單位建立一致的視覺節奏
- **行動版**: 響應式設計，在小螢幕上垂直堆疊控制項，確保觸控友好的按鈕尺寸
